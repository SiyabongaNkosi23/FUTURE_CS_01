## Overview
This project presents a vulnerability assessment conducted on the OWASP Juice Shop web application as part of the Future Interns Cyber Security Internship.

The assessment involved identifying security weaknesses, analyzing vulnerabilities, classifying risk levels, and recommending remediation measures using cybersecurity tools such as Nmap and OWASP ZAP.

---

## Objectives
- Identify common web vulnerabilities
- Perform passive vulnerability scanning
- Analyze exposed services
- Classify risks and findings
- Recommend security improvements

---

## Tools Used
- Nmap
- OWASP ZAP
- Browser Developer Tools
- Canva
- Windows Command Prompt

## Key Findings
- Missing Content Security Policy Header
- Missing X-Frame-Options Header
- Information disclosure risks
- Missing security-related HTTP headers

---

## Recommendations
- Implement secure HTTP headers
- Conduct regular vulnerability assessments
- Apply secure coding practices
- Improve monitoring and security testing
